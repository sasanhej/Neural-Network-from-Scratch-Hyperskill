Description

In the previous stage, we've come up with a bunch of handy functions and rescaled the data. Now, we'll start building our neural network by creating the class and adding the feedforward.

Before you dive into the problem we highly recommend you an interactive tutorial on neural networks by Allison George🧠

First, we will learn the basics by coding the simplest one-layer neural network. In later steps, we will complicate things by adding the second layer. Take a look at the picture below:

It's one layer of a fully connected neural network. It contains no hidden layers; all of the input-layer neurons are connected with neurons of the output layer.

We suggest storing the result of the forward step in a self. variable. It will come in handy once you start working on the backpropagation.

Once you are done with the class, we need to check whether your function works. For that, create an instance of your model with the number of input neurons equal to the number of features you have, and the number of output neurons equal to the number of classes — 10. After that, apply your model to the first two items (= rows) of the training dataset and print the result.

Objectives
  • Create the OneLayerNeural class and implement the __init__ and forward methods inside;
  • Create a class instance with the number of input neurons equal to the number of features and the number of output neurons equal to the number of classes (10);
  • Print the result of the model’s feedforward for the first two items of the training dataset.
