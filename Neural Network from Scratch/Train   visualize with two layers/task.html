Description

All we need to do is to see how well the model performs while training. You can see now why we've recommended the estimator and alpha parameters in the epoch training function in stage 4. If you have followed our advice, you need to use the same training loop you have created previously, only replacing the estimator with our new TwoLayerNeural class model.

In this stage, we want you to train the model with 20 epochs, batch size of 100 and alpha of 0.5, like in stage 4. Don't forget to log the accuracies after each epoch as it's been done previously. You need to print them to pass the tests.

Use X_train, y_train to train your model, and X_test, y_test to calculate the accuracies.

You can use the provided plot function to visualize how the loss and accuracy change while the model is training or you can code it on your own.

Objectives

Put everything together and train your model with 20 epochs, batch size of 100, and alpha of 0.5. Use the functions of stage 4. Print the final list of accuracies after each epoch.
